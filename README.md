# HakoLine_Python
# Hako Line (Python-based Strategic Puzzle Game)

大学一年生時に開発した、PythonとPygameによる対戦型パズルゲームです。
AI（Claude）を開発パートナーとして活用し、**「要件定義・テスト・UI実装」を人間が、「ロジックのベース実装」をAIが担当**するという協調開発フローを実践したプロジェクトです。

## 概要
「囲碁」や「五目並べ」の戦略性に、現代的なスピード感と動的なルール変化を加えた2人対戦型パズルゲームです。

- **独自ルール**: 盤面サイズに応じた勝利条件の変動、タイムアタックモードによる自動指し機能。
- **UI/UX**: AIが苦手とする座標計算やボタン配置を自ら手動でコーディングし、操作性を追求しました。

## 主な機能
- **動的なグリッド生成**: 6x6, 8x8, 10x10への即時切り替え。
- **タイムアタックモード**: 思考時間に制限を設け、時間を過ぎると強制的にランダムな石が置かれる緊張感のあるゲーム性。
- **状態管理システム**: メニュー、プレイ、オプション、リザルトの各状態を効率的に制御。
- **アセット管理**: 画像リソースの非同期ロード（擬似）とエラーハンドリングの実装。

## 技術的アプローチ（AIとの協調開発）
本プロジェクトでは、AIを「単なるコード生成ツール」ではなく「シニアエンジニア」として扱い、以下の分担で進めました。

- **自身の役割（PM/QA/FE）**: 
  - ゲームルールの立案と仕様策定。
  - 実装されたアルゴリズムの動作検証（QA）。
  - AIが苦手とするピクセル単位のUI配置・デザイン調整（フロントエンド実装）。
- **AIの役割（BE/Optimization）**: 
  - 勝利判定や時間計測などのバックエンドロジックの提案。
  - 既存コードのリファクタリング支援。

## 開発環境 / 使用技術
- **Language**: Python 3.10.6
- **Library**: Pygame
- **Tools**: Claude (AI Assistant for Pair Programming)
